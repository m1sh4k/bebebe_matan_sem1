$S_{впис}<S_{сект}<S_{т}$

1. $S_{впис}=\frac{{R^{2}\sin x}}{2}$
2. $S_{сект}=\pi R^{2}\cdot \frac{x}{2\pi}=\frac{xR^{2}}{2}$
3. $S_{т}=\frac{{\mathrm{tg}\ x}}{2}$

$\sin x<x<\mathrm{tg}\ x$

$1< \frac{x}{\sin x}< \frac{1}{\cos x}$

поделим на $\sin x>0$ (при $x \in\left( 0;\frac{\pi}{2} \right)$)

$\cos x<\frac{{\sin x}}{x}<1$

По теореме о зажатой
$$
\left.
\begin{aligned}
\lim_{ x \to 0 } \cos x=1\\
\lim_{ x \to 0 } 1=1\\
\cos x<\frac{{\sin x}}{x}<1\\
\cos x\text{ опр в некторой } \dot{U}_{0}
\end{aligned}
\right\}
\implies 
\lim_{ x \to 0^{+} } \frac{{\sin x}}{x}=1
$$

т.к.  $\frac{{\sin-x}}{-x}=\frac{{\sin x}}{x}$
то функция четная, при $\sin x<0$ аналогично
